# TaoHtml Runtime 路线

TaoHtml Runtime 是嵌入最终 HTML 的通用运行底座。它负责稳定行为，不决定内容和视觉；页面结构、构图和项目专属动效仍由 Agent 生成。

## 当前核心

- 阅读与单屏演讲模式
- 分步动效和整页翻页
- 每页独立动效状态
- 返回页面状态恢复
- 全屏、页码和自动隐藏控制
- 同一编辑模式内修改报告文字、替换图片和拖动裁切焦点
- 文字 / 图片 / 裁切统一撤销重做
- 标签页会话级刷新恢复与导出新 HTML
- 离线资源、hash 路由和浏览器 QA

## 后续可选模块

这些能力只有在实现并通过浏览器测试后，才进入对外能力清单：

- 页面总览
- 双屏演讲者窗口
- 当前页、下一页、口播稿与计时同步
- 交互图表、表格筛选和图表联动
- 跨页连续动效
- 演讲崩溃恢复

## 架构原则

- Runtime 管行为，Agent 管内容和设计。
- 最小核心默认包含；可选模块只在项目需要时加入。
- 不为未使用的功能增加文件体积和运行成本。
- 通用模块通过公开 Runtime API 和状态事件扩展，不替换核心导航。
- 每份成品嵌入其使用的 Runtime 版本，避免交付后因远程更新而失效。
- 完全离线，不依赖 CDN、远程字体或外部脚本。
- 浏览器编辑器只导出新 HTML；含相对 assets 的项目保留原目录结构，不虚假承诺浏览器内 ZIP。
