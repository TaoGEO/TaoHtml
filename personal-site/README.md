# Personal Site Prototype

这是 Taomir / 王涛个人网站的当前原型版本。

## 本地预览

在 Mac 上进入这个目录：

```bash
cd TaoHtml/personal-site
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

## 文件说明

- `index.html`：首页
- `styles.css`：主样式
- `main.js`：首页动效与交互
- `assets/`：肖像与视觉资产
- `DESIGN.md`：内容定位、文案和设计沟通记录
- `style-lab.*`：风格实验页，后续调整审美和动效时使用
